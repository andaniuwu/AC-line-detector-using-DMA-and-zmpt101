# AC-line-detector-using-DMA-and-zmpt101
Project to detect AC line abscence from domestic power net. Control PCB and programming are only to detect ACLine and get information from the BMS to the lithium battery bank control. Peripheral components are: Orion BMS2, CALB lithium battery bank, Quiq 1000 LiCharger, dc/dc, inverter, and control PCB
